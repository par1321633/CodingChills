Array

store items of same type
- index start at 0
Multidimensional Array:
numbers[row_index][col_index]

Applications : lookup tables/hashtables heaps
Dynamic Array
Random Access :- If we know index then complexity O(1)
Fast, Easy

Disadvantages:
Size is fixed, Not so dynamic DS
